# Brutus Objection Page Brief Template

Use this brief before any sub-agent drafts a page.

## Mandatory writing skill

Every draft, hand-written or sub-agent generated, must invoke the `article-writing` skill (`~/.claude/skills/article-writing/SKILL.md`) AND mirror the voice reference at `voice-references/brutus-voice.md`. House style sits on top: no em dashes mid-sentence, no emojis. See `subagent-workflow.md` for the full rule. Drafts that skip the skill, trip its banned patterns, or break the voice reference are rejected on first review.
